# **App Name**: DropFile

## Core Features:

- Access Code Generation: Generate a random, shareable access code for temporary file sharing.
- Access Code Entry: Allow users to join a file sharing session using a provided access code.
- Realtime File List: Display the list of currently shared files in realtime, updating automatically.
- File Upload: Drag-and-drop or select files to upload to the shared space.
- File Auto-Expiration: Automatically expire and delete files after a set period of time.
- AI Access Code Suggestions: Suggest keywords to the user as alternative access codes using an LLM tool to ensure both memorability and uniqueness.

## Style Guidelines:

- Primary color: Light blue (#5DADE2) for a clean, modern, and trustworthy feel.
- Background color: Very light blue (#EBF5FB), almost white, providing a calm backdrop.
- Accent color: Soft orange (#F5B041) for calls to action and important UI elements.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral, and readable interface.
- Use simple, clear icons to represent file types and actions.
- A clean, minimalist layout with a focus on easy file uploading and sharing.
